# Sentiment Analysis of Restaurant Reviews Using NLP

## Project Overview
This project aims to analyze **restaurant customer reviews** using **Natural Language Processing (NLP)** techniques and classify them into **Positive, Negative, or Neutral** sentiments. Using the `restaurant.csv` dataset, the reviews are cleaned and preprocessed before being analyzed.  

The main objectives of the project are:
- **Classify customer reviews** into sentiment categories (Positive, Negative, Neutral).  
- **Evaluate model performance** to ensure reliable sentiment predictions.  
- **Visualize overall trends** in customer feedback through **word clouds** and other visualizations.  
- **Provide actionable insights** to restaurant owners, helping them understand feedback and improve service quality.
